
# Congo Finance - Site d'Investissement Professionnel

Site web professionnel de gestion d'investissements réalisé avec Flask.

## Fonctionnalités
- Design responsive ultra-moderne (Bootstrap 5)
- Sections: Accueil, À propos, Services, Contact
- Animations fluides et micro-interactions
- Formulaire de contact fonctionnel (JavaScript)
- Thème investissement premium (bleu professionnel)

## Installation & Lancement

1. **Installer les dépendances:**
```bash
cd investment_site
pip install -r requirements.txt
```

2. **Lancer le serveur:**
```bash
python app.py
```

3. **Ouvrir le site:**
```
http://127.0.0.1:5000
```
